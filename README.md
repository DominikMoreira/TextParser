# TextParser

TextParser is a command-line tool that analyzes input text using Apple's NaturalLanguage framework. It provides various options including language detection, sentiment analysis, lemmatization, semantic alternatives via embeddings, and named entity recognition.

---

## Requirements

- **macOS** 14 or later (to support the latest NaturalLanguage and NLEmbedding features)  
- **Xcode** 15 or later (safe baseline)  
- **Swift** 5.9 or compatible  
- The project builds as a command line tool  

---

## Build Instructions

### Using Xcode

1. Open the project in Xcode.  
2. Select the command line tool target.  
3. Build and run.
